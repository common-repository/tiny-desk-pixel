It helps you add easily your tracking tag from Tiny Desk by just giving your advertiser id and pixel id.

== Description ==

With this plugin we will help you set up your tracking tag (pixel) on your website without the need of developer knowledge.

Install it, go to the settings page and give us your Advertiser Id and Pixel Id. That will set up your tracking tag code on the <head> of your site.

Why a pixel is important?

It will help you track which ads are working and which ones are not, to provide the best ad experience to your potential customers.

== Installation ==

1. Upload `tiny-desk-pixel` folder to the `/wp-content/plugins/` directory of your wordpress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on 'Tiny Desk Pixel' under the 'Settings' menu in WordPress
4. Type your Advertiser Id, Pixel Id and click 'Save Changes'
5. And you are done.
